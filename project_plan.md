# Children's Joyland — Plan de proiect

## 1. Descriere proiect
Site de prezentare **one-page** pentru grădinița/centrul educațional **Children's Joyland**. Publicul țintă: părinți cu copii de vârstă preșcolară. Scopul principal: **programarea unei vizite**. Scopuri secundare: înțelegerea atmosferei, valorilor, ofertei educaționale și grupelor disponibile.

## 2. Structură pagină (one-page, anchor navigation)
- `/` — Pagina unică cu secțiuni ancorate:
  - `#acasa` — Hero
  - `#despre-noi` — Despre noi
  - `#oferta` — Oferta educațională
  - `#o-zi` — O zi la Joyland
  - `#grupe` — Grupe
  - `#taxe` — Taxe
  - `#galerie` — Galerie
  - `#contact` — Contact + formular programare vizită
